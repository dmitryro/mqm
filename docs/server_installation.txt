Install a new server
====================

Install these packages::

    subversion
    git
    python
    python-dev
    ipython
    mysql-server
    python-mysqldb
    python-imaging
    daemontools

Put the following into /etc/init/svscan.conf::

    # svscan - daemontools
    #
    # This service starts daemontools from the point the system is
    # started until it is shut down again.

    start on startup

    start on runlevel 1
    start on runlevel 2
    start on runlevel 3
    start on runlevel 4
    start on runlevel 5
    start on runlevel 6

    stop on shutdown

    exec svscanboot

And run::

    sudo start svscan
